# VRCX-Luo Web

VRCX-Luo 的 Cloudflare Pages 静态介绍页。

VRCX-Luo 基于 [FuLuTang/VRCX-jirai](https://github.com/FuLuTang/VRCX-jirai) 继续修改，部分修复、整理和功能补充由 AI 辅助完成。该直接来源分支长期未继续更新，所以这里作为个人维护版继续补齐日常使用需要。VRCX-jirai 源自 [vrcx-team/VRCX](https://github.com/vrcx-team/VRCX)。

## Cloudflare Pages 部署

- Framework preset: `None`
- Build command: 留空
- Build output directory: `/`

页面入口是 `index.html`，可直接作为静态站点部署。
